# MiniShop-Catalog
Bài tập PHP - MiniShop Catalog

## Chức năng

- Hiển thị danh mục sản phẩm
- Hiển thị danh sách sản phẩm
- Lọc sản phẩm theo danh mục
- Sử dụng dữ liệu từ data.php

## Công nghệ

- PHP
- HTML
- CSS

## Cách chạy

1. Copy project vào thư mục htdocs của XAMPP
2. Khởi động Apache
3. Truy cập
